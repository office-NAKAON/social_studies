# ポートフォリオサイト

シンプルでモダンなポートフォリオサイトのテンプレートです。  
HTML・CSS・JavaScriptのみで構成されているため、そのままブラウザで開いて確認できます。

## 構成

- **index.html** … メインのHTML（Hero / About / Skills / Projects / Contact）
- **styles.css** … デザイン・レイアウト
- **script.js** … モバイルメニュー・スクロール時の挙動

## 使い方

1. **ローカルで確認する**
   - `index.html` をダブルクリックしてブラウザで開く
   - または、VS Code の「Live Server」などでローカルサーバーを起動して開く

2. **内容を編集する**
   - `index.html` 内の「あなたの名前」「自己紹介」「スキル」「プロジェクト」「連絡先」などを自分の情報に書き換える
   - プロジェクト画像は `.project-image` 内に `<img>` を入れるか、背景画像として指定可能

3. **デザインを変える**
   - `styles.css` の先頭の `:root` 内で色・フォント・角丸などを変更できる
   - 例: `--color-accent: #2563eb;` を別の色にすると全体のアクセント色が変わる

## カスタマイズのヒント

- **タイトル・メタ情報**: `<title>` と `<meta name="description">` を自分の名前・説明に変更
- **プロジェクト**: カードを増減したり、画像・リンクを追加
- **Contact**: `mailto:` や SNS のURLを自分のものに変更
- **公開**: GitHub Pages、Netlify、Vercel などにアップロードすると無料で公開できます

## ライセンス

自由に改変・利用してください。
